





# ProConnect | Professional Networking Platform

A full-stack professional networking application featuring a React-based terminal aesthetic and a robust Java Spring Boot backend.

## 🛠️ Technical Stack
* **Frontend:** React.js, Vite, Tailwind CSS
* **Backend:** Java Spring Boot, Spring Data JPA
* **Database:** MySQL (Relational Schema)
* **Communication:** RESTful API with Asynchronous Polling logic

## 🚀 How to Run
1.  **Backend:** Open the Spring Boot project in STS and run as a Spring Boot App (Port 8080).
2.  **Database:** Ensure MySQL is running with the `proconnectdb` schema.
3.  **Frontend:** Run `npm install` and `npm run dev` in the client folder.
